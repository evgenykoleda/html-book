Разделители

Разделители используются для для формирования структуры текста документа.

Перенос строки

В HTML разметке перенос строки не приводит к переносу строки в итоговом представлении. Для этого существует одиночный тег <br>. Перенос строки как правило используется для отделения предложений в стихах и текстах песен. Ошибочно использовать тег <br> для отделения абзацев, для этой цели используйте тег <p>.

Разделительная линия

Одиночный тег <hr> позволяет создавать горизонтальную разделительную линию. Таким образом можно разделять разные части текста, например главны стихов.
