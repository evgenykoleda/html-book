Списки

Списки позволяют создавать перечисления, как одиночные так и ассоциативные. Как правило элементами перечисления являются объекты предыдущего абзаца или предложения.

Оглавление

  1. Маркированные списки
  2. Упорядоченные списки
  3. Списки определений

Маркированные списки

Позволяют создавать одиночные неупорядоченные перечисления. Список начинается открывающимся тегом <ul>, каждый элемент списка добавляется в виде вложенного парного тега <li>. Заканчивается список закрывающимся тегом </ul> Элемент списка может содержать вложенный список точно таким-же образом.

Пример: 

<ul>
    <li>
        Птицы
    </li>
    <li>
        Рыбы       
        <ul>
            <li>
                Окунь
            </li>
            <li>
                Щука
            </li>
            <li>
                Плотва
            </li>
        </ul>
    </li>
    <li>
        Млекопитающие
    </li>
</ul>

Результат:

  • Птицы
  • Рыбы
    • Окунь
    • Щука
    • Плотва
  • Млекопитающие

Упорядоченные списки

Позволяют создавать одиночные упорядоченные перечисления. Способ разметки идентичен маркированному списку: открывающий тег списка <ol>, тег элемента списка <li>. Каждый элемент списка увеличивает свой номер на еденицу относительно предыдущего.

Атрибуты:

start
Номер первого элемента списка. По умолчанию равен 1.

Списки определений

Позволяют создавать ассоциативные неупорядоченные перечисления. Каждому элементу перечисления соответствует ассоциация в виде его определения. Список начинается открывающим тегом <dl>, каждый элемент задается парой вложенных тегов <dt> и <dd>. С помощью тега <dt> задается термин, с помощью <dd> его определение.

Пример: 

<dl>
    <dt>HTML</dt>
    <dd>Hypertext Markup Language - язык гипертекстовой разметки</dd>
    
    <dt>CSS</dt>
    <dd>Cascading style sheets - каскадные таблицы стилей</dd>
</dl>

Результат:

HTML
Hypertext Markup Language - язык гипертекстовой разметки
CSS
Cascading style sheets - каскадные таблицы стилей
