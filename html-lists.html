Списки

Списки позволяют создавать перечисления, как одиночные так и ассоциативные. Как правило элементами перечисления являются объекты предыдущего абзаца или предложения.

Маркированные списки

Позволяют создавать одиночные неупорядоченные перечисления. Список начинается открывающимся тегом <ul>, каждый элемент списка добавляется в виде вложенного парного тега <li>. Заканчивается список закрывающимся тегом </ul> Элемент списка может содержать вложенный список точно таким-же образом.
Пример маркированного списка, в который вложен другой маркированный список:

<ul>
    <li>
        Птицы
    </li>
    <li>
        Рыбы       
        <ul>
            <li>
                Окунь
            </li>
            <li>
                Щука
            </li>
            <li>
                Плотва
            </li>
        </ul>
    </li>
    <li>
        Насекомые
    </li>
</ul>

Упорядоченные списки

Позволяют создавать одиночные упорядоченные перечисления. Способ разметки идентичен маркированному списку: открывающий тег списка <ol>, тег элемента списка <li>. Каждый элемент списка увеличивает свой номер на еденицу относительно предыдущего. По умолчанию номер первого элемента равен 1, аттрибут start позволяет задать произвольный номер.
